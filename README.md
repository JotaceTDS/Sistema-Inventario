Sistema Corporativo de Inventário, Usuários e Acessos
Sistema desenvolvido em C# (WinForms) com MySQL, focado em organizar operações internas da empresa:  
Inventário de equipamentos, cadastro de pessoas, controle de linhas moveis, departamentos, fornecedores e acessos a plataformas externas.
O sistema continua em crescimento conforme demanda da empresa.
